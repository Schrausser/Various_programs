Various programs

Tools for calculations, bootstrapping and permutation tests, also for practice purposes when learning programming languages ​​(C, Basic, Fortran etc.), see e.g. Halvorson and Rygmyr (1991), Chivers and Sleightholme (2018), Joyce (2019), Streib (2020) or Gonzalez-Morris and Horton (2024):

Mathematics:

Potenzen, decconv, Nl, winkel, fx_euler, sinus_sum, kugeloberflaeche, kugelraum_1, kugelvolumen, kugelvolumen_n1, kugelvolumen_n2, tetra1, tetra_alg1, tetra_alg2, tetra_h, tetra_h1, tetra_o, tetra_s, tetrah, tetrah1, tetrao, tetrao1, tetrav, tetrav1, z00, z001.
